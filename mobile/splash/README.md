# Hair & Makeup Pro - Splash Screens

iOS splash screens for the add-to-homescreen experience.

## Required Sizes

Create splash screens at these dimensions:

- splash-640x1136.png (iPhone 5)
- splash-750x1334.png (iPhone 6/7/8)
- splash-1242x2208.png (iPhone 6/7/8 Plus)
- splash-1125x2436.png (iPhone X/XS)
- splash-1170x2532.png (iPhone 12/13/14)

## Design Guidelines

- Use the app's gold color (#C9A961) as background
- Center the app icon and name
- Keep it simple and fast-loading
